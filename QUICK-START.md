# دليل البدء السريع

## خطوات التثبيت السريع

### 1. تثبيت Dependencies

```bash
cd quality-tickets-system/api
npm install
```

### 2. إعداد قاعدة البيانات

```bash
# نسخ ملف الإعدادات
cp ENV_TEMPLATE.env .env

# تعديل .env وإضافة بيانات قاعدة البيانات
nano .env
```

### 3. تهيئة قاعدة البيانات

```bash
npm run init-db
```

### 4. تشغيل الخادم

```bash
npm start
```

### 5. فتح المتصفح

اذهب إلى: `http://localhost:3001`

## الحسابات الافتراضية

- **المدير**: `admin / admin123`
- **موظف الجودة**: `quality / quality123`

## الخطوات التالية

1. سجل دخول كمدير
2. أنشئ حسابات للفرق
3. ابدأ بإدخال التكتات من واجهة موظف الجودة

## ملاحظات مهمة

- تأكد من أن MySQL يعمل
- تأكد من أن المنفذ 3001 متاح
- الصور تُحفظ في مجلد `uploads/`

## استكشاف الأخطاء

### خطأ في الاتصال بقاعدة البيانات
- تحقق من إعدادات `.env`
- تأكد من أن MySQL يعمل

### خطأ في رفع الصور
- تأكد من وجود مجلد `uploads/`
- تحقق من الصلاحيات

### الصفحة لا تعمل
- تحقق من أن الخادم يعمل
- افتح Console في المتصفح للتحقق من الأخطاء
